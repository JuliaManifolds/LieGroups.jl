LieGroups.AbstractLeftGroupActionType ‚Äî Type
AbstractLeftGroupActionType <: AbstractGroupActionType

A type representing a (smooth) group action $œÉ: \mathcal G √ó \mathcal M ‚Üí \mathcal M$ of a LieGroup $\mathcal G$ acting (from the left) on an AbstractManifold $\mathcal M$. with the following properties

  1. $œÉ(\mathrm{e}, p) = p$ holds for all $p ‚àà \mathcal M$
  2. $œÉ(g, œÉ(h, p)) = œÉ(g‚àòh, p)$ holds for all $g,h ‚àà \mathcal G$, $p ‚àà \mathcal M$

where $‚àò$ denotes the group operation of the LieGroup $\mathcal G$. See also [HN12, Definition 9.1.11].

The type of action can be seen a bit better when writing the action as a family $œÉ_g(p)$: we obtain from the second property as

\[ œÉ_g(œÉ_h(p)) = œÉ_{gh}(p)\]

and see that $g$ appears on the left.

To emphasize the side the group operation is acting from, we sometimes write $œÉ^{\mathrm{L}}$. If the action is clear from context we write $œÉ(g, p) = g ‚ãÖ p$.

One notable example of a left action is the inverse of an action of AbstractRightGroupActionType $œÉ^{\mathrm{R}}$, which is given by $œÑ_g = (œÉ^{\mathrm{R}}_g)^{-1} = œÉ^{\mathrm{R}}_{g^{-1}}$. We obtain

\[œÑ_g(œÑ_h(p)) = œÉ^{\mathrm{R}}_{g^{-1}}(œÉ^{\mathrm{R}}_{h^{-1}}(p)) = œÉ^{\mathrm{R}}_{h^{-1}g^{-1}}(p) = œÉ^{\mathrm{R}}_{(gh)^{-1}}(p) œÑ_{gh}(p).\]

Note

In function definitions we follow the idea of the family of actions and use the order (M, g, p) in function signatures.

AbstractRightGroupActionType <: AbstractGroupActionType

A type representing a (smooth) group action $œÉ: \mathcal M √ó \mathcal G ‚Üí \mathcal M$ of a LieGroup $\mathcal G$ acting (from the right) on an AbstractManifold $\mathcal M$. with the following properties

  1. $œÉ(p, \mathrm{e}) = p$ holds for all $p ‚àà \mathcal M$
  2. $œÉ(œÉ(p, g), h) = œÉ(p, g‚àòh)$ holds for all $g,h ‚àà \mathcal G$, $p ‚àà \mathcal M$

where $‚àò$ denotes the group operation of the LieGroup $\mathcal G$. See also [HN12, Remark 9.1.12].

The type of action can be seen a bit better when writing the action as a family $œÉ_g(p)$: we obtain from the second property as

\[ œÉ_g(œÉ_h(p)) = œÉ_{hg}(p)\]

and see that $g$ appears on the right.

To emphasize the side the group operation is acting from, we sometimes write $œÉ^{\mathrm{R}}$. If the action is clear from context we write $œÉ(p, g) = p ‚ãÖ g$.

One notable example of a right action is the inverse of an action of AbstractLeftGroupActionType $œÉ^{\mathrm{L}}$, which is given by $œÑ_g = (œÉ^{\mathrm{L}}_g)^{-1} = œÉ^{\mathrm{L}}_{g^{-1}}$. We obtain

\[œÑ_g(œÑ_h(p)) = œÉ^{\mathrm{L}}_{g^{-1}}(œÉ^{\mathrm{L}}_{h^{-1}}(p)) = œÉ^{\mathrm{L}}_{g^{-1}h^{-1}}(p) = œÉ^{\mathrm{L}}_{(hg)^{-1}}(p) œÑ_{hg}(p).\]

Note

In function definitions we follow the idea of the family of actions and use the order (M, g, p) in function signatures.

LieGroups.GroupAction ‚Äî Type
GroupAction{T<:GroupActionType, L<:LieGroup, M<:AbstractManifold}

Specify a group action of AbstractGroupActionType T of a LieGroup G acting on M.

Let $\mathcal M$ be a AbstractManifold and $\mathcal G$ be a LieGroup with group operation $‚àò$.

A (smooth) action of the group $\mathcal G$ on the manifold $\mathcal M$ is a map

\[œÉ: \mathcal G √ó \mathcal M ‚Üí \mathcal M\]

with the properties

  • $œÉ(\mathrm{e}, p) = p$ holds for all $p ‚àà \mathcal M$
  • $œÉ(g, œÉ(h, p)) = œÉ(g‚àòh, p)$ holds for all $g,h ‚àà \mathcal G$, $p ‚àà \mathcal M$

Fields

  • type::T: The type of the group action.
  • group::L: The group acting.
  • manifold::M: The manifold the group acts upon.

See [HN12, Section 9.1.3] for more details.

LieGroups.InverseLeftGroupOperationAction ‚Äî Type
InverseLeftGroupOperationAction <: AbstractRightGroupActionType

A type for the AbstractLeftGroupActionType when acting on the group itself given by the inverse of a LeftGroupOperationAction $œÉ_h$ as

\[œÑ_h(g) \coloneqq œÉ_h^{-1}(g) = œÉ(h^{-1},g) = h^{-1}‚àòg\]

Note that while in naming it is the inverse of the left action, it's properties yield that is is an AbstractRightGroupActionType, since

\[œÑ_g(œÑ_h(p)) = œÉ^{\mathrm{L}}_{g^{-1}}(œÉ^{\mathrm{L}}_{h^{-1}}(p)) = œÉ^{\mathrm{L}}_{g^{-1}h^{-1}}(p) = œÉ^{\mathrm{L}}_{(hg)^{-1}}(p) œÑ_{hg}(p).\]

for its inverse $(œÉ_h)^{-1}$ see InverseLeftGroupOperationAction.

Note

Some literature also calls this by itself the right group operation action.
