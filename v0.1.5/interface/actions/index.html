LieGroups.AbstractLeftGroupActionType ‚Äî Type
AbstractLeftGroupActionType <: AbstractGroupActionType

A type representing a (smooth) group action $œÉ: \mathcal G √ó \mathcal M ‚Üí \mathcal M$ of a AbstractLieGroup $\mathcal G$ acting (from the left) on an AbstractManifold $\mathcal M$. with the following properties

  1. $œÉ(\mathrm{e}, p) = p$ holds for all $p ‚àà \mathcal M$
  2. $œÉ(g, œÉ(h, p)) = œÉ(g‚àòh, p)$ holds for all $g,h ‚àà \mathcal G$, $p ‚àà \mathcal M$

where $‚àò$ denotes the group operation of the AbstractLieGroup $\mathcal G$. See also [HN12, Definition 9.1.11].

The type of action can be seen a bit better when writing the action as a family $œÉ_g(p)$: we obtain from the second property as

\[ œÉ_g(œÉ_h(p)) = œÉ_{gh}(p)\]

and see that $g$ appears on the left.

When writing about general group actions, the symbol $Œ±$ is often used. The order of arguments then follows the same as the one of the left action. Most often we use the index notation $Œ±_g(p)$. The definition of functions also follows this notation, i.e. we use e.g. apply(A, g, p)

One notable example of a left action is the inverse of an action of AbstractRightGroupActionType $œÑ$, which is given by $œÉ_g = (œÑ_g)^{-1} = œÑ_{g^{-1}}$. We obtain

\[œÉ_g(œÉ_h(p)) = œÑ_{g^{-1}}(œÑ_{h^{-1}}(p)) = œÑ_{h^{-1}g^{-1}}(p) = œÑ_{(gh)^{-1}}(p) œÉ_{gh}(p).\]

AbstractRightGroupActionType <: AbstractGroupActionType

A type representing a (smooth) group action $œÑ: \mathcal M √ó \mathcal G ‚Üí \mathcal M$ of a AbstractLieGroup $\mathcal G$ acting (from the right) on an AbstractManifold $\mathcal M$. with the following properties

  1. $œÑ(p, \mathrm{e}) = p$ holds for all $p ‚àà \mathcal M$
  2. $œÑ(œÑ(p, g), h) = œÑ(œÑ(p, g), h)$ holds for all $g,h ‚àà \mathcal G$, $p ‚àà \mathcal M$

where $‚àò$ denotes the group operation of the AbstractLieGroup $\mathcal G$. See also [HN12, Remark 9.1.12].

The type of action can be seen a bit better when writing the action as a family $œÑ_g(p)$: we obtain from the second property as

\[ œÑ_g(œÑ_h(p)) = œÑ_{hg}(p)\]

and see that $g$ appears on the right.

When writing about general group actions, the symbol $Œ±$ is often used. In that case the order of arguments follows either the one from the left action, but most often we use the index notation. The definition of functions also follows this notation, i.e. we use e.g. apply(A, g, p)

One notable example of a right action is the inverse of an action of AbstractLeftGroupActionType $œÉ$, which is given by $œÑ_g = (œÉ_g)^{-1} = œÉ_{g^{-1}}$. We obtain

\[œÑ_g(œÑ_h(p)) = œÉ_{g^{-1}}(œÉ_{h^{-1}}(p)) = œÉ_{g^{-1}h^{-1}}(p) = œÉ_{(hg)^{-1}}(p) œÑ_{hg}(p).\]

LieGroups.ActionActsOnLeft ‚Äî Type
ActionActsOnLeft <: AbstractActionActsOnType

An AbstractActionActsOnType representing that an action acts on the left.

This is meant in the following way: Given a GroupAction $Œ±: \mathcal G √ó \mathcal H ‚Üí \mathcal H$ where a Lie group $\mathcal G$ acts on another Lie group $\mathcal H$ with an arbitrary action.

Then, e.g. within the definition of the LeftSemidirectProductGroupOperation or RightSemidirectProductGroupOperation, we have two choices where the group action $Œ±$ acts on, namely: Let $g ‚àà \mathcal G$ and $h_1,h_2 ‚àà \mathcal H$ be given, then this type represents the variant

\[Œ±_g(h_1) ‚ãÖ h_2,\]

where $‚ãÖ$ denotes the group operation on $\mathcal H$. The Left in the name of this type refers to the fact that the action is applied to the left element $h_1$.

Note that this is independent of both the type of action (left or right) and whether the semidirect product is a left or a right semidirect one.

For its practical use see the SemidirectProductGroupOperation.

Then, e.g. within the definition of the LeftSemidirectProductGroupOperation or RightSemidirectProductGroupOperation, we have two choices where the group action $Œ±$ acts on, namely: Let $g ‚àà \mathcal G$ and $h_1,h_2 ‚àà \mathcal H$ be given, then this type represents the variant

LieGroups.GroupAction ‚Äî Type
GroupAction{T<:GroupActionType, L<:LieGroup, M<:AbstractManifold}

Specify a group action of AbstractGroupActionType T of a AbstractLieGroup G acting on an AbstractManifold M.

Let $\mathcal M$ be a AbstractManifold and $\mathcal G$ be a AbstractLieGroup with group operation $‚àò$.

A (smooth) action of the group $\mathcal G$ on the manifold $\mathcal M$ is a map

\[Œ±: \mathcal G √ó \mathcal M ‚Üí \mathcal M\]

with the properties

Identity. $Œ±(\mathrm{e}, p) = p$ holds for all $p ‚àà \mathcal M$

Compatibility. If $Œ±$ is a $AbstractLeftGroupActionType$ we usually denote it by $œÉ$ and the compatibility reads

\[œÉ_g(œÉ_h(p)) = œÉ_{g‚àòh}(p) \text{ holds for all} g,h ‚àà \mathcal G \text{ and} p ‚àà \mathcal M\]

If $Œ±$ is a $AbstractRightGroupActionType$ we usually denote it by $œÑ$ and the compatibility reads

\[œÑ_g(œÑ_h(p)) = œÑ_{h‚àòg}(p)`` holds for all ``g,h ‚àà \mathcal G \text{ for all} p ‚àà \mathcal M\]

intuitively the left/right property of an action specifies on which side the ‚Äúouter‚Äù group actions element $g$ gets ‚Äúappended‚Äù in the composition.

Fields

  • type::AbstractGroupActionType: The type of the group action.
  • group::AbstractLieGroup: The group acting.
  • manifold::AbstractManifold: The manifold the group acts upon.

See [HN12, Section 9.1.3] for more details.

LieGroups.InverseLeftGroupOperationAction ‚Äî Type
InverseLeftGroupOperationAction <: AbstractRightGroupActionType

A type for the AbstractLeftGroupActionType when acting on the group itself given by the inverse of a LeftGroupOperationAction $œÉ_h$ as

\[œÑ_h(g) \coloneqq œÉ_h^{-1}(g) = œÉ(h^{-1},g) = h^{-1}‚àòg\]

Note that while in naming it is the inverse of the left action, it's properties yield that is is an AbstractRightGroupActionType, since

\[œÑ_g(œÑ_h(p)) = œÉ_{g^{-1}}(œÉ_{h^{-1}}(p)) = œÉ_{g^{-1}h^{-1}}(p) = œÉ_{(hg)^{-1}}(p) œÑ_{hg}(p).\]

for its inverse $(œÉ_h)^{-1}$ see InverseLeftGroupOperationAction.

Note

Some literature also calls this by itself the right group operation action.
